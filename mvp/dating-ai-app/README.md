# AI缘分助手 MVP

AI智能交友匹配平台 - 找到最适合的另一半

## ✨ 功能特点

- 🤖 **AI智能匹配**：基于性格、兴趣的精准匹配算法（92%高匹配度）
- 📷 **照片上传**：支持上传最多6张个人照片展示
- 💝 **详细资料**：个人介绍、兴趣爱好、择偶标准完整填写
- 💬 **实时聊天**：一对一聊天功能，支持文字消息
- 💡 **AI破冰话题**：智能生成4条开场白建议
- 📍 **约会推荐**：基于兴趣推荐约会地点（4类场景）
- 🔒 **隐私设置**：照片、位置、在线状态、消息权限管控
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 16 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **状态管理**: React Hooks
- **构建工具**: Turbopack

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3005
