# AI缘分助手 MVP

AI智能交友匹配平台

## ✨ 功能特点

- 🤖 **AI智能匹配**：基于性格、兴趣的精准匹配算法
- 💝 **真实认证**：实名认证，保障交友安全
- 💬 **高效沟通**：AI破冰话题，轻松开启聊天
- 📊 **匹配度评分**：直观展示匹配指数
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3000
