# 宠物健康助手 MVP

AI智能宠物健康管理平台

## ✨ 功能特点

- 🏥 **AI症状分析**：智能分析宠物症状，提供初步健康评估
- 💊 **健康建议**：根据症状提供专业护理和就医建议
- 📋 **健康记录**：完整的宠物健康档案管理
- 📱 **移动端优化**：完美适配手机和平板设备
- 🎨 **精美设计**：温暖的琥珀色主题

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **AI分析**: 模拟智能分析算法

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3000
