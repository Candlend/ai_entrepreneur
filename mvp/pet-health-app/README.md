# 宠物健康助手 - 完整版

AI智能宠物健康管理平台 - 全方位呵护您的爱宠

## ✨ 功能特点

### 核心功能（8大模块）
1. **👤 宠物档案** - 完整的宠物信息管理，包含基本信息和统计数据
2. **🏥 健康记录** - AI症状分析，智能健康评估和建议
3. **🍖 喂食提醒** - 定时喂食提醒，自动通知功能
4. **💊 用药记录** - 详细的用药管理，包含剂量、频率、周期
5. **🏥 就诊记录** - 医院就诊历史，诊断结果和治疗方案
6. **📷 照片相册** - 记录宠物成长的美好瞬间
7. **🎯 成长里程碑** - 时间轴展示重要成长时刻
8. **💰 支出记录** - 详细的养宠支出统计和分析

### 技术亮点
- 8个功能标签页，完整的宠物管理系统
- AI驱动的健康分析和建议
- 时间轴可视化展示
- 支出统计和分类
- 响应式设计，完美适配移动端
- 温暖的琥珀色主题设计

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **AI分析**: 模拟智能分析（可接入Claude API）

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3002 查看应用

### 构建生产版本
```bash
npm run build
npm start
```

## 📖 使用指南

### 1. 宠物档案
- 查看宠物基本信息（名字、品种、年龄、体重）
- 显示各项数据统计（健康记录、用药、照片、支出）
- 快速了解宠物整体状况

### 2. 健康记录
- 记录宠物症状和不适
- AI智能分析健康状况
- 提供专业护理建议
- 历史健康记录查询

### 3. 喂食提醒
- 设置每日喂食时间
- 记录食物类型和分量
- 开启/关闭提醒功能
- 多个时间段提醒

### 4. 用药记录
- 记录药品名称和剂量
- 设置用药频率
- 标记用药周期（开始/结束日期）
- 便于长期用药管理

### 5. 就诊记录
- 记录医院和医生信息
- 详细的就诊原因和诊断
- 治疗方案存档
- 医疗费用统计

### 6. 照片相册
- 上传宠物照片
- 添加照片说明
- 按时间排序展示
- 记录成长瞬间

### 7. 成长里程碑
- 时间轴展示重要时刻
- 记录特殊日子（生日、纪念日）
- 添加成长事件
- 可视化成长历程

### 8. 支出记录
- 记录各类养宠支出
- 分类统计（食品、玩具、医疗、美容）
- 总支出自动计算
- 支出分析和建议
