# AI视频剪辑助手 MVP

基于 AI 的智能视频剪辑应用 - 自动识别高光时刻

## ✨ 功能特点

- 🎬 **智能视频分析**：AI自动识别视频中的精彩时刻和场景切换
- ✂️ **一键剪辑**：自动生成30秒/1分钟/3分钟短视频
- 📤 **拖拽上传**：支持MP4、MOV、AVI、MKV格式，最大2GB
- 📱 **移动端优化**：完美适配手机和平板设备
- 🎨 **精美设计**：现代渐变配色、流畅动画

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **文件上传**: react-dropzone
- **视频处理**: FFmpeg (计划中)
- **AI分析**: Claude Vision API (计划中)

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看应用
