# AI视频剪辑助手 - 完整版

基于 AI 的智能视频剪辑应用 - 自动识别高光时刻，一键生成专业短视频

## ✨ 功能特点

### 核心功能（8大模块）
1. **📤 视频上传** - 拖拽上传，支持MP4、MOV、AVI、MKV格式，最大2GB
2. **✂️ 视频剪裁** - 可视化时间轴，精确裁剪开始/结束时间
3. **🎵 添加音乐** - 内置音乐库，4种风格背景音乐可选
4. **📝 智能字幕** - AI语音识别自动生成字幕（Whisper技术）
5. **🎨 视觉效果** - 4种转场效果 + 6种滤镜效果
6. **⚡ 速度调整** - 0.25x - 4x速度调节，支持慢动作和快进
7. **🎯 AI高光分析** - 自动识别精彩时刻和场景切换
8. **💾 多格式导出** - MP4视频/时间轴JSON/Premiere工程文件

### 技术亮点
- 8个功能标签页，清晰的工作流程
- AI驱动的高光时刻检测和场景分割
- 实时预览和参数调整
- 响应式设计，完美适配移动端
- 现代渐变UI设计

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **文件上传**: react-dropzone
- **AI分析**: Claude Vision API（计划中）
- **语音识别**: Whisper API（计划中）
- **视频处理**: FFmpeg（计划中）

## 📖 使用指南

### 1. 上传视频
- 拖拽视频文件到上传区域
- 或点击按钮选择本地视频文件
- 支持格式：MP4、MOV、AVI、MKV
- 文件大小限制：2GB

### 2. 视频剪裁
- 拖动滑块调整开始时间和结束时间
- 实时显示剪辑后的时长
- 支持精确到秒的剪裁

### 3. 添加音乐
- 从音乐库中选择背景音乐
- 4种风格：欢快、温馨、史诗、电子
- 音乐自动适配视频长度

### 4. 添加字幕
- 点击"AI自动识别"按钮
- 使用Whisper技术自动转写音频
- 可手动编辑每条字幕文本
- 显示字幕时间轴

### 5. 视觉效果
- **转场效果**：淡入淡出、滑动、缩放、溶解
- **滤镜效果**：无滤镜、复古、黑白、鲜艳、冷色调、暖色调

### 6. 速度调整
- 拖动滑块或点击预设速度
- 支持范围：0.25x - 4x
- 实时显示效果预览

### 7. AI高光分析
- 自动检测视频中的精彩时刻
- 场景分割和切换检测
- 显示每个高光的置信度和时间
- 音频峰值检测

### 8. 导出视频
- 查看完整的剪辑配置总览
- 选择导出格式：
  - MP4视频（通用格式）
  - 时间轴JSON（供进一步编辑）
  - Premiere工程文件（专业剪辑软件）
- 一键导出

- 基于帧差异的场景检测

- Whisper语音识别引擎
- 自动断句和时间戳
- 支持中英文识别
- 可编辑的时间轴
