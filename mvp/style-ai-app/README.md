# AI穿搭助手 - 完整版

AI智能穿搭推荐平台 - 专业搭配，展现最佳风采

## ✨ 功能特点

### 核心功能（6大模块）
1. **📸 AI分析** - 智能识别身材、肤色和个人风格，提供专业分析
2. **👔 衣橱管理** - 管理您的衣物，记录类别、颜色、季节、场合
3. **🎯 场合推荐** - 正式、休闲、约会等不同场合的穿搭建议
4. **🌸 季节推荐** - 春夏秋冬四季穿搭方案
5. **🏷️ 品牌推荐** - 奢侈、中端、平价三档品牌推荐
6. **✨ 搭配示例** - 视觉化搭配展示和搭配技巧

### 技术亮点
- 6个功能标签页，完整的穿搭管理系统
- AI驱动的风格分析
- 多维度推荐（场合、季节、品牌）
- 衣橱管理功能
- 响应式设计，完美适配移动端
- 时尚的粉紫渐变主题

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **文件上传**: react-dropzone
- **AI分析**: 模拟智能分析（可接入Claude Vision API）

## 📖 使用指南

### 1. AI分析
- 上传您的照片（全身照效果更佳）
- AI自动识别身材类型、肤色色调
- 获取个性化风格推荐
- 查看专属配色方案

### 2. 衣橱管理
- 添加衣物到虚拟衣橱
- 记录衣物属性（类别、颜色、季节、场合）
- 统一管理所有服装
- 方便快速查找和搭配

### 3. 场合推荐
根据不同场合提供专业穿搭建议：
- **正式场合**：西装套装、连衣裙、精致配饰
- **休闲场合**：T恤牛仔裤、针织衫、运动风
- **约会场合**：优雅连衣裙、修身衬衫、浪漫配饰

### 4. 季节推荐
针对四季提供穿搭方案：
- **春季**：薄外套、T恤、针织衫
- **夏季**：清爽上衣、连衣裙、短裤
- **秋季**：风衣、夹克、毛衣
- **冬季**：羽绒服、大衣、厚外套

### 5. 品牌推荐
三档品牌满足不同预算：
- **奢侈品牌**：Gucci、Prada、Chanel
- **中端品牌**：Zara、H&M、Uniqlo
- **平价品牌**：优衣库、网易严选、淘宝精选

### 6. 搭配示例
- 视觉化搭配展示
- 实用搭配技巧
- 颜色搭配建议
- 配饰选择指南

## 💡 穿搭技巧

### 身材修饰
- **梨形身材**：上宽下窄，突出上半身
- **苹果型身材**：避免腰部紧身，选择A字型
- **沙漏型身材**：展现曲线，腰部收紧
- **H型身材**：制造曲线，层次搭配

### 色彩搭配
- **冷色调肤色**：适合粉色、蓝色、紫色
- **暖色调肤色**：适合黄色、橙色、红色
- **中性色调**：各种颜色都适合

### 场合原则
- **正式场合**：简洁大方，精致得体
- **休闲场合**：舒适自在，个性展现
- **约会场合**：优雅浪漫，展现魅力
