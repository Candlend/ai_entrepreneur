# 流量变现工具

AI智能优化广告投放，最大化网站流量变现收益的专业平台。

## 功能特性

### 核心功能
- **数据概览**：实时展示总收益、曝光量、点击量、平均CTR
- **广告位管理**：管理多个广告位，查看详细数据和表现
- **AI优化建议**：基于数据分析提供个性化优化建议
- **收益预测**：预测优化后的收益提升空间
- **趋势分析**：7天收益趋势可视化展示
- **热门排行**：自动识别表现最好的广告位

### 技术亮点
- 三大功能模块（数据概览/广告位/优化建议）
- 响应式设计，完美适配移动端和桌面端
- 实时数据更新和统计
- 可视化图表展示
- 智能优化建议系统
- 优雅的紫粉渐变UI设计

## 使用指南

### 1. 数据概览
- 查看关键指标（收益、曝光、点击、CTR）
- 查看7天收益趋势图
- 查看热门广告位排行
- 切换时间周期（今日/本周/本月）

### 2. 广告位管理
- 查看所有广告位详细数据
- 编辑广告位配置
- 添加新广告位
- 批量操作（刷新/导出/设置）

### 3. 优化建议
- 查看AI生成的优化建议
- 查看预期收益提升
- 执行优化方案
- 跟踪优化进度

## 数据指标说明

- **曝光量**：广告被展示的次数
- **点击量**：用户点击广告的次数
- **CTR**：点击率（点击量/曝光量）
- **CPM**：千次曝光收益
- **填充率**：广告成功填充的比例

## 技术栈

- **框架**：Next.js 15 (App Router)
- **语言**：TypeScript, React 19
- **样式**：Tailwind CSS v4
- **特性**：
  - Client-side state management
  - 响应式布局 (sm/md breakpoints)
  - 渐变背景和玻璃态效果
  - 可视化数据图表
  - SVG图标和动画

## MVP特性

当前版本为演示MVP，使用模拟数据展示核心功能：
- 4个模拟广告位
- 7天收益趋势数据
- 4条AI优化建议
- 完整的数据统计和分析
- 移动端优化

## 优化建议类型

- **成功**：表现优秀的广告位，建议扩大规模
- **警告**：需要改进的问题点，建议立即处理
- **信息**：市场趋势和机会，建议关注
