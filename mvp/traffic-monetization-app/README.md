# 流量变现工具 MVP

AI智能优化广告投放 - 实时数据更新 · 多平台聚合 · A/B测试

## ✨ 功能特点

- 📊 **实时数据更新**：自动刷新数据，30秒一次更新
- 📍 **广告位管理**：管理多个广告位，查看详细表现
- 🚀 **AI优化建议**：基于数据分析提供个性化优化方案
- 🧪 **A/B测试功能**：对比不同版本，选择最优方案
- 🔗 **多平台聚合**：集成Google AdSense、百度联盟等5大平台
- 📄 **报表导出**：支持Excel/PDF/CSV格式导出
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 16 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **状态管理**: React Hooks + useEffect
- **构建工具**: Turbopack

## 🚀 快速开始

```bash
npm install
npm run dev
```

## 数据指标说明

- **曝光量（Impressions）**：广告被展示的次数
- **点击量（Clicks）**：用户点击广告的次数
- **CTR（Click-Through Rate）**：点击率 = 点击量/曝光量
- **CPM（Cost Per Mille）**：千次曝光收益
- **填充率（Fill Rate）**：广告成功填充的比例
