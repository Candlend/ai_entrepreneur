# AI求职顾问 MVP

AI智能求职辅导平台

## ✨ 功能特点

- 🎤 **模拟面试**：真实场景演练，5道经典面试题
- 📝 **简历优化**：AI分析简历，提供专业建议
- 🎯 **职业规划**：定制化职业发展路径
- 📊 **AI评估**：综合评分和逐题反馈
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 15 + React 19 + TypeScript
- **样式**: Tailwind CSS v4

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3000
