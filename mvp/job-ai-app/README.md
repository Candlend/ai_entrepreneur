# AI求职顾问 MVP

AI智能求职辅导平台 - 提供全方位求职服务

## ✨ 功能特点

- 🎤 **模拟面试**：真实场景演练，5道经典面试题，AI即时反馈
- 📝 **简历优化**：AI深度分析，提供评分、优势、弱点和优化建议
- 🎯 **职业规划**：3阶段发展路径，技能树可视化
- 💰 **薪资查询**：行业薪资范围、影响因素和谈判建议
- 🏢 **公司信息**：企业文化、福利待遇、面试建议
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 16 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **状态管理**: React Hooks
- **构建工具**: Turbopack

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3004

## 🎯 功能详情

### 1. 🎤 模拟面试（已完成 ✅）
- 5道经典面试题覆盖：基础问题、自我认知、能力评估、动机考察、职业发展
- 进度条实时显示面试进度
- STAR法则（Situation-Task-Action-Result）答题指导
- 实时字数统计
- AI即时反馈和评分（总体85分，逐题评分75-95分）
- 综合评估报告展示优势和改进建议
- 面试完成后可重新开始

### 2. 📝 简历优化（已完成 ✅）
- 支持简历文件上传（PDF/DOC/DOCX格式）
- 支持粘贴简历文本内容
- AI分析中加载动画（2秒模拟分析）
- 综合评分（0-100分制，带可视化进度条）
- 优势亮点分析（4项具体优势）
- 需要改进分析（4项改进点）
- 详细优化建议（5条可执行建议）
- 关键词提取（技能标签可视化展示）
- 支持重新分析新简历

### 3. 🎯 职业规划（已完成 ✅）
- 职业目标输入和分析
- AI生成3阶段职业发展路径
- 时间轴可视化展示（1-2年、3-4年、5年+）
- 每阶段所需技能标签清晰展示
- 当前职位突出显示
- 阶段间视觉连接线效果
- 支持重新规划

### 4. 💰 薪资查询（已完成 ✅）
- 职位薪资范围查询（如：15K - 40K）
- 市场平均薪资数据展示
- 5大关键影响因素详细分析：
  - 工作年限
  - 所在城市
  - 公司规模
  - 技能栈
  - 学历背景
- 4条实用薪资谈判建议
- 双列网格布局展示核心数据
- 支持查询多个职位对比

### 5. 🏢 公司信息（已完成 ✅）
- 公司基本信息（名称、行业、规模）
- 员工评分（5星评价可视化）
- 公司文化详细描述
- 福利待遇标签展示（六险一金、弹性工作、年度体检等）
- 4条针对性面试建议
- 支持查询多家公司对比
