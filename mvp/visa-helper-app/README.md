# 美签预约助手 MVP

AI智能监控美国签证预约系统 - 多渠道通知 · 自动预约 · 材料清单

## ✨ 功能特点

- 🔍 **实时监控**：7x24小时监控美国使领馆签证预约系统
- 🔔 **三重通知**：邮件、短信、微信多渠道即时提醒
- ⚡ **自动预约**：发现可用时间后自动完成预约
- 📋 **预约历史**：完整记录所有预约状态和时间
- 📄 **材料清单**：详细的签证材料准备清单和提示
- 💡 **预约技巧**：8大实用技巧提高预约成功率
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 16 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **状态管理**: React Hooks
- **构建工具**: Turbopack

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3006
