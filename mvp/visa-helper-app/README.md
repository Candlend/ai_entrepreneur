# 美签预约助手

AI智能监控美国签证预约系统，第一时间发现可用时间并自动预约。

## 功能特性

### 核心功能
- **实时监控**：7x24小时监控美国使领馆签证预约系统
- **智能提醒**：邮件、短信、微信多渠道即时通知
- **自动预约**：发现可用时间后自动完成预约
- **多城市支持**：覆盖北京、上海、广州、成都、沈阳等主要城市
- **全签证类型**：支持F-1学生签、B-1/B-2旅游商务签、H-1B工作签等

### 技术亮点
- 响应式设计，完美适配移动端和桌面端
- 直观的三步配置流程
- 实时状态显示（可预约/名额紧张/已满）
- 优雅的渐变UI设计
- 智能算法预测最佳预约时间

## 使用指南

### 1. 配置监控
- 选择使领馆城市
- 选择签证类型
- 填写通知邮箱
- 决定是否启用自动预约

### 2. 监控运行
- 系统每5分钟自动检查一次
- 实时显示可用预约时间
- 查看最近的通知记录

### 3. 预约确认
- 点击"立即预约"快速预约
- 或等待自动预约完成

## 技术栈

- **框架**：Next.js 15 (App Router)
- **语言**：TypeScript, React 19
- **样式**：Tailwind CSS v4
- **特性**：
  - Client-side state management
  - 响应式布局 (sm/md breakpoints)
  - 渐变背景和玻璃态效果
  - SVG图标和动画

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

npm start
```

访问 http://localhost:3000 查看应用
