# AI声乐教练

AI智能分析唱功，提供专业声乐指导和个性化训练的教学平台。

## 功能特性

### 核心功能
- **学习进度追踪**：完整的学习数据统计和进度可视化
- **AI录音分析**：实时分析音准、节奏、音色、气息等维度
- **智能评分系统**：综合评分和分项评分，精准定位问题
- **个性化建议**：基于分析结果提供针对性改进建议
- **专业课程体系**：从基础到高级的系统化训练课程
- **练习曲目库**：精选热门歌曲，支持伴奏播放

### 技术亮点
- 四大功能模块（首页/练习/分析/课程）
- 响应式设计，完美适配移动端和桌面端
- 实时录音状态显示
- 可视化数据分析图表
- 分级难度系统（初级/中级/高级）
- 优雅的靛紫渐变UI设计

## 使用指南

### 1. 首页
- 查看学习进度和统计数据
- 查看今日推荐练习
- 快速开始训练

### 2. 录音练习
- 点击开始录音按钮
- AI实时分析声音
- 录音完成后查看分析

### 3. 唱功分析
- 查看综合得分和排名
- 查看各维度详细评分
- 获取个性化改进建议
- 推荐相关训练课程

### 4. 课程学习
- 浏览全部训练课程
- 查看课程详情和效果
- 跟随学习路径系统学习

## 分析维度

- **音准**：演唱音高的准确度
- **音准稳定性**：音高保持的稳定程度
- **节奏感**：节拍和节奏的准确性
- **音色**：声音的质感和饱满度
- **气息控制**：呼吸和气息的运用

## 课程体系

### 阶段一：基础入门
- 音阶练习
- 呼吸控制训练
- 发声方法

### 阶段二：技巧提升
- 颤音练习
- 转音技巧
- 咬字吐字

### 阶段三：音域拓展
- 高音突破
- 低音训练
- 音域扩展

### 阶段四：情感表达
- 情感处理
- 舞台表现
- 演唱完整性

## 技术栈

- **框架**：Next.js 15 (App Router)
- **语言**：TypeScript, React 19
- **样式**：Tailwind CSS v4
- **特性**：
  - Client-side state management
  - 响应式布局 (sm/md breakpoints)
  - 渐变背景和玻璃态效果
  - 可视化分析图表
  - 动画和过渡效果
  - SVG图标

## 评分标准

- **85-100分**：优秀（绿色）
- **70-84分**：良好（黄色）
- **0-69分**：需改进（红色）
