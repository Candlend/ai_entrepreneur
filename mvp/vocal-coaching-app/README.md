# AI声乐教练 MVP

AI智能分析音准节奏 - 实时检测 · 作品分享 · PK对战

## ✨ 功能特点

- 🎯 **实时音准检测**：录音时显示动态音准波形图
- 🎤 **录音播放**：支持播放/暂停，进度条展示
- 🔌 **API集成**：支持5种主流语音识别服务
- 📤 **作品分享**：一键分享到微信、抖音、微博
- 📅 **打卡系统**：记录连续天数，统计练习数据
- ⚔️ **PK对战**：与好友比拼唱功，实时得分对比
- 📱 **移动端优化**：完美适配各种设备

## 🛠 技术栈

- **前端框架**: Next.js 16 + React 19 + TypeScript
- **样式**: Tailwind CSS v4
- **状态管理**: React Hooks + useEffect + useRef
- **构建工具**: Turbopack

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3008
