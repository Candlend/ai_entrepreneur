# 健康管理助手 - 完整增强版

AI驱动的健康管理应用，提供食物识别、体重追踪、运动记录、体脂分析等全方位健康管理功能。

## ✨ 核心功能（已完整实现）

### 1. 🍽️ 智能饮食管理
- **AI食物识别**：拍照自动识别食物并计算营养信息
- **热量追踪**：实时显示今日热量摄入 vs 目标
- **营养分析**：详细的蛋白质、碳水化合物、脂肪数据
- **饮食日志**：记录每餐食物，可删除管理
- **智能提醒**：根据摄入情况显示剩余热量

### 2. 🏃 运动记录系统
- **运动追踪**：记录运动类型、时长、消耗热量
- **运动统计**：今日运动总时长、总消耗、活动次数
- **运动分类**：有氧运动、力量训练、柔韧性训练
- **可视化展示**：运动数据图表展示
- **快速添加**：一键添加常见运动项目

### 3. ⚖️ 体重管理
- **体重记录**：每日体重记录和历史查看
- **趋势图表**：可视化体重变化趋势
- **目标追踪**：显示体重变化和目标进度
- **数据分析**：周/月体重变化统计

### 4. 📊 AI体脂率分析（创新功能）
- **智能识别**：上传身体照片，AI分析体型
- **体脂估算**：预估体脂率范围（误差±3%）
- **内脏脂肪**：评估内脏脂肪等级
- **个性化建议**：基于分析结果提供健康建议
- **免责声明**：明确标注仅供参考

### 5. 📈 数据统计分析
- **周度统计**：7天热量摄入趋势图
- **营养分析**：蛋白质/碳水/脂肪摄入完成度
- **目标进度**：健康目标完成百分比
- **对比分析**：摄入 vs 消耗数据对比

### 6. 👤 个人信息与设置
- **基础信息**：年龄、性别、身高、体重
- **BMR计算**：基础代谢率（Mifflin-St Jeor方程）
- **TDEE计算**：每日总能量消耗
- **目标设置**：减重/维持/增重目标
- **活动水平**：5档活动强度选择
- **智能推荐**：基于个人数据的目标热量

## 📋 使用说明

### 饮食管理
1. 点击"🍽️ 饮食"标签
2. 拍照或选择食物照片
3. 点击"开始识别"，AI自动识别
4. 查看营养信息和今日总计
5. 可删除误添加的记录

### 运动记录
1. 点击"🏃 运动"标签
2. 点击"+ 添加运动"
3. 输入运动名称、时长、消耗
4. 查看今日运动统计

### 体重追踪
1. 点击"⚖️ 体重"标签
2. 点击"+ 记录体重"
3. 输入当前体重
4. 查看趋势图和历史记录

### 体脂分析
1. 点击"📊 体脂"标签
2. 上传身体照片（正面/侧面）
3. 点击"开始分析"
4. 查看体脂率和健康建议

### 数据统计
1. 点击"📈 统计"标签
2. 查看周度热量趋势
3. 查看营养摄入分析
4. 了解目标完成进度

### 个人设置
1. 点击"👤 设置"标签
2. 查看个人基本信息
3. 查看BMR和TDEE数据
4. 查看目标热量设置
