# 瘦享AI - Landing Page

减肥健康管理应用的产品展示页面，用于收集用户预订单。

## 技术栈

- Next.js 15 + React 19
- Tailwind CSS v4
- TypeScript

## 核心功能

- 产品价值主张展示
- 核心功能介绍
- 预订单表单收集
- 响应式设计

## 本地开发

```bash
cd apps/health-landing
pnpm install
pnpm dev
```

访问 http://localhost:3000

## 部署

使用 Vercel 一键部署：

```bash
vercel
```

## 目标

- 至少收集 30 个预订单
- 转化率 > 5%
